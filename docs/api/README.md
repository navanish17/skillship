# API reference

Live Swagger UI:  http://localhost:8000/api/docs/
Live ReDoc:       http://localhost:8000/api/redoc/
Raw OpenAPI JSON: http://localhost:8000/api/schema/

We do NOT hand-write an API doc. The schema is generated by `drf-spectacular` from the DRF serializers + views.
To regenerate the TypeScript client types for the frontend:

```bash
cd frontend && npm run gen:types
```
